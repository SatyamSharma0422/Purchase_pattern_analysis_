# 🛒 Purchase Pattern Analysis (Market Basket Analysis)

## 📊 Overview
This project analyzes customer purchasing behavior using Market Basket Analysis to identify relationships between products. The goal is to improve cross-selling strategies and increase revenue through data-driven insights.

## 🔧 Tools Used
- SQL
- Power BI
- Excel
- Apriori Algorithm

## 📁 Dataset
- 22,000+ transactions
- 4,298 customers

## 📈 Key Insights
- Customer retention rate: 65.5%
- Identified strong product associations (Lift up to 20.31)
- Medium basket sizes (3–4 items) show highest retention
- High-value customers contribute majority of revenue

## 💡 Business Recommendations
- Implement product bundling strategies
- Target medium basket customers for retention campaigns
- Improve cross-selling using association rules
